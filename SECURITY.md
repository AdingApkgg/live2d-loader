# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.x     | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in live2d-loader, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email the maintainer directly or use [GitHub's private vulnerability reporting](https://github.com/AdingApkgg/live2d-loader/security/advisories/new).

### What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response timeline

- **Acknowledgment**: within 48 hours
- **Initial assessment**: within 1 week
- **Fix release**: as soon as possible, depending on severity

## Scope

This policy covers the `@live2d-loader/*` npm packages. Note that the Cubism SDK cores (`live2d.min.js`, `live2dcubismcore.min.js`) are third-party proprietary software loaded from CDN at runtime and are outside the scope of this project's security policy.
